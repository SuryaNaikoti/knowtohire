# Release Notes: v0.7-marketplace-production

Release v0.7 marks the launch of the **Marketplace & Payment Systems** on **KnowToHire**.

## 🚀 Features Delivered

1. **ATS-Ready Resume Marketplace**:
   - Digital template galleries and categories filters.
   - Dynamic search capabilities.
2. **Billing History Logs**:
   - Customer payment receipts and invoices list.
3. **Upgrade Plans**:
   - Subscription upgrade, cancellation, and renewal workflows.
4. **Checkout Engine**:
   - Unified Stripe/PayPal simulated payment gateway checkouts.

## 🗄️ Database Changes
- Added tables `template_categories`, `templates`, `subscription_plans`, `orders`, `order_items`, `template_purchases`, `payments`, `payment_events`, `subscriptions`, `invoices`.
- Registered triggers and applied RLS locks.
- Regenerated TypeScript type definition mappings.
