### 🛡️ RLS Policies
Row Level Security applied to all tables:

- `profiles` — users can read/write own profile only
- `candidates` — candidate data scoped to auth user
- `employers` — employer data scoped to auth user
- `jobs` — employers manage own jobs; public read for published
- `applications` — candidates see own; employers see for their jobs
- `team_members` — scoped to employer
- `locations` — scoped to employer

### 🔄 Schema Reconciliation
