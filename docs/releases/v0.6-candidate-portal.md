# Release v0.6-candidate-portal

This release delivers a fully operational candidate portal, verified schema migrations, RLS coverage, and layout updates.

## 📋 Features Delivered
- **Interactive Portfolio Settings**: Candidate biography, headlines, privacy, and salary configurations.
- **Unified Upload Engine**: Secure file uploads for resumes (`resumes` bucket) and profile pictures (`avatars` bucket).
- **Comprehensive Candidate Subsections**: Education History, Verified Certifications, and Portfolio Projects with full CRUD capabilities.
- **Notifications Hub**: Grouped timeline notifications with mark-as-read updates.
- **Job Alerts**: Personalized search alerts with daily/weekly/monthly email frequency toggles.
- **Public Profile Layout**: Shareable candidate profile url at `/profile/:candidateId`.
- **Dynamic Headers**: Live unread notification indicator badge in the header layout.
- **Instant Job Applications**: Directly apply for positions from Job Details pages.

## 💾 Database Changes
- Migration `20260705000001_enhance_candidate_projects.sql` — Alters `candidate_projects` table to add `tech_stack` and `is_featured` columns.
- Migration `20260705000002_enhance_job_alerts.sql` — Alters `job_alerts` table to add `is_active` and `frequency` columns.
