# Release Notes: v0.8 Content CMS & Analytics

Version 0.8 introduces a comprehensive Content Management System (CMS), Lead Magnet gating system, behavioral analytics, security audit logging, comment threads, and community-driven resource requests.

## Key Features

### 1. Blog / Content CMS
- Paginated blog listing with categories and full-text search.
- Clean SEO-friendly design utilizing Inter and Outfit typography.
- Atomic view-tracking via Postgres RPC function `increment_blog_post_view_count`.
- Full post detail viewer with HTML rendering sanitized by DOMPurify to eliminate XSS risks.
- Admin management screen to easily add, update, publish, or unpublish posts.

### 2. Lead Magnets
- Interactive lead generation board supporting gated content (checklists, e-books, templates).
- Flexible gating options: Email-only (CRM capture) or Account-required.
- Atomic download counting using PostgreSQL RPC function `increment_lead_magnet_download_count`.
- Automatically tracks download captures along with UTM parameters (source, medium, campaign) for marketing analysis.

### 3. Resource Requests
- Community request board where users can request guides, templates, and courses.
- Duplicate-safe upvoting logic managed by unique database constraints.
- Real-time upvote sync using an automated database trigger `trigger_sync_upvote_count` on upvote insert/delete.

### 4. Behavioral Analytics
- Fire-and-forget user analytics tracking page views, blog opens, upvotes, and lead magnet downloads.
- Daily aggregation engine ready to drive performance tracking dashboards.
- Admin stats panel displaying Top Visited Pages, Event breakdown, and Event timelines.

### 5. Admin Audit Logging
- Security audit logs capturing major mutations, logins, and administrative actions.
- Automatic logs written with customizable severity ratings (info, warning, error, critical).

## Database Migrations
- `20260706000001_content_cms_system.sql` — Schema setup for CMS, requests, captures, and analytics.
- `20260706000002_sprint7_hardening.sql` — Security updates, atomic RPC functions, indexes, and sync triggers.

## Performance & Hardening
- Added 6 index strategies for high-frequency queries (`idx_blog_posts_published_deleted`, `idx_analytics_events_created`, etc.).
- Embedded `DOMPurify` to sanitize HTML content outputs in BlogPostDetail page.
- Enabled Row Level Security (RLS) on all 8 tables with proper admin validation subqueries.
