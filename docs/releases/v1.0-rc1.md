# Release Candidate 1 (v1.0-rc1) Release Notes

Version 1.0-rc1 establishes feature completeness for KnowToHire, transitioning the codebase from active feature development to production readiness testing.

## Key Changes
- **Feedback Widget:** Mounts a feedback capture component globally in public beta mode.
- **CI/CD Integration:** Integrates `.github/workflows/deploy.yml` pipeline validating build tests and executing remote migrations auto-pushes on main merge cycles.
- **Documentation:** Complete operations playbooks, on-call manuals, and disaster recovery processes.
