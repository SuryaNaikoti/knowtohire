# Release Candidate 1 (v1.0-rc1) Plan

This plan details release validation criteria for moving from `v1.0-beta` to Release Candidate status.

## Verification Checklist

1. **Regression Suite Runs**
   - Execute all verification scenarios across candidate, employer, and billing modules.

2. **Subdomain Resolvers Check**
   - Confirm routing parameters match correctly for tenant URLs.

3. **Stripe Billing Integrity**
   - Sandbox payment checks on active subscription tiers.

4. **AI Pipeline Telemetry**
   - Verify performance latency on semantic matching queries (<100ms).
